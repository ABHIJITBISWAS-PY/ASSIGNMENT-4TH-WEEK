Description ofTidy data set

variables

Tidy data set contains 180 rows and 68 columns. the tidy data set with the average of each variable for each activity and each subject.

mean(): Mean value
std(): Standard deviation

Subsymbol column is numbered sequentially from 1 to 30 and the activity column has 6 types as listed below.

WALKING
WALKING_UPSTAIRS
WALKING_DOWNSTAIRS
SITTING
STANDING
LAYING
